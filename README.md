# Gemini Garage Repair (Coral Springs)

A modern, responsive React-based website for a local garage door repair business.

## Tech Stack
- **React** (v18)
- **TypeScript**
- **Tailwind CSS**
- **Vite**
- **Lucide React** (Icons)

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
